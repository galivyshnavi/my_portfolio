Project Overview

The Semma Coffee Shop website was designed to showcase a local coffee business, featuring their menu, location, and contact information. The website is fully responsive and provides an engaging user experience for coffee enthusiasts.

Homepage of the coffee shop website

Key Features

  • Fully responsive design for all devices
  • Interactive menu with category filtering
  • Google Maps integration for location
  • Contact form with validation
  • Smooth scrolling navigation
  • Mobile-friendly hamburger menu
  • Performance optimized for fast loading

Interactive menu section with filtering options

Technical Details

The website was built using vanilla HTML, CSS, and JavaScript without any frameworks to ensure fast loading times and broad browser compatibility.

Key technical aspects include:

  • CSS Flexbox and Grid for layout
  • Form validation with JavaScript
  • Responsive images with srcset
  • Accessibility best practices

Challenges & Solutions

Challenge: Creating a responsive menu that works well on both desktop and mobile.

Solution: Implemented a mobile-first design with a hamburger menu that transforms into a horizontal navigation on larger screens.
